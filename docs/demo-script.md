# 演示脚本

这份脚本用于评审或答辩时快速复现核心功能，避免现场从空日历开始手动造数据。

## 一键演示数据

打开应用后点击“演示数据”，系统会在当前访客或已登录账号下写入 5 条样例日程：

- 1 分钟后的喝水提醒，用于观察页面浮窗、助手反馈和语音播报。
- 今天或明天下午的演示复盘，用于展示今日列表。
- 明天上午团队周会，用于展示添加、查看和修改。
- 下一个周五产品评审，用于展示未来日程和月历事件标记。
- 三天后的健身课，用于展示月历事件标记。

重复点击“演示数据”会先移除旧的 `demo-` 样例，再生成新的样例；用户自己添加的日程不会被覆盖。

## 推荐演示流程

1. 点击“演示数据”，查看全部列表、今日列表和月历标记。
2. 输入“查看今天日程”，验证范围查询。
3. 输入“把明天团队周会改到下午四点”，验证修改日程。
4. 输入“取消明天团队周会”，再按候选提示确认删除。
5. 等待 1 分钟，验证喝水提醒是否出现页面浮窗、助手反馈和语音播报。
